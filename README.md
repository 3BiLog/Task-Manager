# Task Manager

Ứng dụng Android đơn giản được xây dựng bằng Kotlin và Jetpack Compose để hiển thị màn hình hoàn thành nhiệm vụ.

## Giới thiệu

Đây là project luyện tập Jetpack Compose giúp làm quen với:
- Hiển thị hình ảnh
- Hiển thị văn bản
- Căn giữa nội dung
- Sử dụng `Column` và `Modifier`
- Thiết kế giao diện theo Material Design

Ứng dụng hiển thị:
- Biểu tượng hoàn thành nhiệm vụ
- Tiêu đề thông báo
- Nội dung trạng thái công việc

## Công nghệ sử dụng

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio

## Kiến thức đã học

- `@Composable`
- `Image`
- `Text`
- `Column`
- `Modifier`
- `fillMaxSize()`
- `padding()`
- `Alignment.CenterHorizontally`
- `Arrangement.Center`
- `FontWeight.Bold`
- `painterResource()`
- `Surface`
- `MaterialTheme`
